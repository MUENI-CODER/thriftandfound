THRIFT & FOUND WEBSITE
======================

Folder Structure:
- index.html - Homepage
- shop.html - Clothing shop page
- jobs.html - Job openings listing
- apply.html - Job application form
- about.html - About the shop
- contact.html - Contact page
- css/style.css - All styling
- js/main.js - Main functionality
- js/apply.js - Application form handling
- images/ - Add your own images here

To Add Images:
1. Place images in /images/clothing/ folder
2. Update image paths in inventory array (main.js)

To Make Forms Actually Send Email:
You'll need a backend (PHP, Node.js, or a service like Formspree)

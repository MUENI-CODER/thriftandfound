# Thrift & Found Website

A complete thrift shop website with job application system.

## Features

- Product catalog with categories
- Dynamic pricing (prices decrease daily)
- Job application form with CV upload
- Admin panel to manage applications
- Contact form
- Mobile responsive design

## Live Website

[thriftandfound.infinityfreeapp.com](http://thriftandfound.infinityfreeapp.com)

## Admin Access

- URL: `/admin-login.html`
- Username: `admin`
- Password: `admin123`

## Technologies Used

- HTML5, CSS3, JavaScript
- PHP for backend
- JSON for data storage
- InfinityFree hosting
